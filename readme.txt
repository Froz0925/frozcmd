Froz scripts
============

Windows 10+ 64 bit.

Использование:
В папке bin должны быть необходимые exe-утилиты.
1. В проводнике: удерживая Ctrl, выделите файлы и перетащите на CMD-файл.
   Или скопируйте пути и вставьте в командную строку.  
2. В FAR / Total Commander: Ctrl+Enter - вставить имя под курсором,
   В FAR: выделите файлы клавишей Ins, затем Ctrl+G, введите:
   sound-all2wav "!.!"
